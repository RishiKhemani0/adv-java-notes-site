## Definition
- These are classes that will be used to represent a table in the database.
- Each object can be used to represent a row in that table
- All models follow builder pattern
- All variables are final
## BLOBS
These are a special type used to represent any datatype with raw bytes, this is read as byte[]
**Only Present in MySQL**
